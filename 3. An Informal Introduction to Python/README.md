## 3.1. 파이썬을 계산기로 사용하기

1. 일반 사칙계산이 가능하고 괄호로 묶는것도 가능합니다.

<pre><code>
>>> 2+2
4
>>> (50- 5*6)/4
5.0
</code></pre>

나눗셈(**/**)은 항상 **float**을 돌려줍니다.
정수 나눗셈 결과(몫)을 얻으려면 **//** 연산자를 사용합니다.

2. 거듭제곱을 계산할 때는 **`**`\*\* 을 사용합니다.

<pre><code>
>>> 3 ** 2
9
</code></pre>
